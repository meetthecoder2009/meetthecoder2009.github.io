Project 1: Climate Action Visualizer

This project visualizes global challenges related to climate change and sustainability. Through shapes and colors, I represented the message of action, nature, and responsibility. It aligns with global goals such as climate action, life on land, and responsible consumption.

What I learned
  • Using multiple geometric shapes in one canvas
  • Applying fill colors to shapes
  • Using gradients as a background
  • Using labels with styling (size, bold)
  • Organizing elements visually with coordinates
  • Aligning shapes symmetrically
Rect(0,0,200,200, fill=gradient('deepSkyBlue','oldLace','darkOrange', start='left'))
Label('Climate Action', 100, 25, size=15, bold=True)
Polygon(30,200,28,195,33,185,48,105,43,180, fill='peru')
Star(48,105,40,9, fill='limeGreen')
Star(48,105,30.8, fill='green')
Polygon(158,200,148,195,153,185,168,105,163,180, fill='black')
Line(168,114,146,116)
Line(166,126,185,130)
Polygon(70,160,70,170,115,170,115,160)
RegularPolygon(118,165,15,3, rotateAngle=90)
What the code does

This code creates a tree using shapes and colors, including a brighter version and a darker version to portray how changes in the climate can impact the trees.

Project 2: Climate & Contrast

This project explores the contrast between a healthy and an unhealthy environment. On the left, a vibrant tree symbolizes climate action and life on land. On the right, a decayed tree and chaotic elements represent what happens when we don’t take care of our planet. The project visualizes the impact of our actions using symmetry, color theory, and shape composition.

What I learned
  • Creating symmetrical designs using coordinates
  • Using gradients to add mood and emotion
  • Layering different shapes to build objects
  • Demonstrating environmental contrast through visual elements
  • Positioning text labels effectively
  • Using complex shapes like stars and polygons in combination

What the code does

This code divides the canvas into two sides: a healthy green tree (left) and a lifeless black tree (right). The gradient background shows a blend from fresh blue to fiery orange, representing hope versus danger. The left tree is made of brown trunks and green stars as leaves, while the right tree is black with jagged lines symbolizing decay. The label at the top sets the theme.
