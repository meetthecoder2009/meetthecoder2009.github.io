Project 1: Climate Action Visualizer

This project visualizes global challenges related to climate change and sustainability. Through shapes and colors, I represented the message of action, nature, and responsibility. It aligns with global goals such as climate action, life on land, and responsible consumption.

This creates a tree with color and a darker version too.

What I learned:

  • How to use multiple geometric shapes in one canvas
  • How to apply fill colors to shapes
  • How to use gradients as a background
  • How to use labels with styling (size, bold)
  • How to organize elements visually with coordinates
  • How to layer shapes for visual depth
  • How to represent symbolic ideas (like a tree)
  • How to combine stars, polygons, and lines
  • How to embed projects using iframes
  • How to align shapes symmetrically
