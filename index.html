Project 4: Mission Possible: Zero Hunger Quiz

"Mission Possible: Zero Hunger Quiz" is an interactive educational game designed to raise awareness about global hunger and food insecurity. Users answer multiple-choice questions, receive immediate feedback, and learn interesting facts related to each topic. The quiz aims to highlight the causes, effects, and potential solutions to hunger, encouraging players to think about how they can contribute to a world with zero hunger.

What I learned
  • Timing and user feedback.
  • Function design and reusability
  • Using variables to track progress
  • Creating and updating UI elements
  • Event handling with mouse input
  • Conditional logic using if statements
What the code does

This code controls the behavior of a multiple-choice quiz app by checking the user's answer based on which question they are on. When the user clicks on one of the answer choices, the program checks if the click matches the correct label for that specific question—label3 for questions 2 and 4, and label2 for question 6. If the user selects the correct answer, a green “Correct!” message is shown, their score (app.correct) increases by one, and the app moves on to the next question. If the user selects a wrong answer, a red “Incorrect, try again” message appears, but the question does not advance. The questionLayout function is used to display the question text, while optionLayout sets up the four answer choices as clickable labels. Overall, this code helps manage question flow, user interaction, and feedback within the quiz.

Project 5: Truck Delivery Game

In this project, I created a food delivery truck simulation where the player picks up food from green stations and drops it off at purple stations. Use the arrow keys to move and press ENTER to interact.

What I learned
  • Collision detection between shapes and sprites
  • Updating game state through keyboard events
  • Managing visibility of stations after delivery
  • Displaying dynamic labels to track game stats
What the code does

This code sets up green pick-up and purple drop-off stations and uses collision detection (hitsShape) to manage deliveries. When the truck hits a station and ENTER is pressed, food is either picked up or dropped off, and the station visibility changes. Labels track the current food load and truck capacity.

greenstation = Rect(50,325,50,50, fill='green')
purplestation1 = Rect(240,318,50,50, fill='purple', visible=True)
purplestation2 = Rect(60,180,50,50, fill='purple', visible=True)

truckFood = Label(app.truckfood,370,20, visible=False)

def interactwithStation(station, amount):
    if truck.hitsShape(greenstation):
        if app.truckfood < app.truckcapacity:
            app.truckfood += amount
            truckFood.value = app.truckfood
    if truck.hitsShape(purplestation2):
        if app.truckfood > 0 and app.truckfood <= app.truckcapacity:
            app.truckfood -= amount
            truckFood.value = app.truckfood
            purplestation2.visible = False
    elif truck.hitsShape(purplestation1):
        if app.truckfood > 0 and app.truckfood <= app.truckcapacity:
            app.truckfood -= amount
            truckFood.value = app.truckfood
            purplestation1.visible = False

def onKeyHold(keys):
    if truck.hitsShape(purplestation1) and 'enter' in keys:
        interactwithStation(purplestation1, 20)
    if truck.hitsShape(purplestation2) and 'enter' in keys:
        interactwithStation(purplestation2, 30)
    if truck.hitsShape(greenstation) and 'enter' in keys:
        interactwithStation(greenstation, 50)
